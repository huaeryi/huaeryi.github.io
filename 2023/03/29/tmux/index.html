Usage

ctrl-b c 新建窗口
ctrl-b p 前一个窗口
ctrl-b n 下一个窗口
ctrl-b % 垂直拆分窗口
ctrl-b " 水平拆分窗口
ctrl-b o 在窗口间切换
exit 退出tmux
