Links

  • lab download

Intro

  • datalab主要是使用位运算来实现函数，考验对底层数据类型的掌握
  • 修改bit.c文件以彰显你的位运算技术
  • 32bit

Solution

bitXor

  • 用位与和位非实现位异或
  • 德摩根律走起

tmin

  • 要求返回最小的补码值0x10000000

isTmax

  • 判断是否是补码最大值
  • 由tmax的特征可得，特判0xFFFFFFFF

allOddBits

  • 要求判断所有奇数位是否都为1
  • 先取奇数位
  • 再判断是否与0xAAAAAAAA相同

negate

  • 求一个数的负数
  • ~x + 1取反加一，常见的转换了

isAsciiDigit

  • 判断是否是Ascii码的0-9，0x30 <= x <= 0x39
  • 分三个条件
  • 前26位为0
  • 形如0x3*
  • 最后四位减去10，符号位为1

conditional

  • 实现?运算符
  • 1取反加一为0xFFFFFFFF，0取反加一再取反为0x0，通过掩码实现选择

isLessOrEqual

logicalNeg

  • 实现逻辑非!

howManyBits

  • 通过二的倍数不断逼近，得到所需位数

floatScale2

  • 实现IEEE浮点数*2
  • 对非规格化，
  • 对NaN返回本身
  • 对计算后expr超过255的返回正负无穷
  • 否则expr++

floatFloat2Int

  • 实现浮点数转int

floatPower2

  • 实现2.0^x的IEEE浮点数
